## 格式要点

- 标题：纯文本 `YYYY.MM.DD《剑侠世界4》稳定性汇总`（加粗、#）
- 平台标题：`*一、PC共12台设备，其中{N}台执行超过4小时。**`（平台名和序号加粗）
- 设备数：`共**N台**设备`（数字、台加粗）
- 执行时长：`其中**M台**执行超过4小时`（数字、台加粗）
- 版本行：`- （版本：v...）`
- 任务行：`  - 1.[任务名](URL) 任务执行汇总：共**N台**设备，其中**M台**设备执行超过4小时，{异常判断}`（数字、台加粗）
- 异常设备缩进：`    - 设备名@IP执行了时长，卡死原因 [系统日志](链接) | [游戏日志](链接) | [Perfeye](链接)`
- 无异常：`未发现异常`
- 有异常：`发现**X台**异常`（数字、台加粗）
- 链接分隔符：` | `（竖线两侧有空格）

## 字段说明

| 字段 | 说明 |
|------|------|
| `{YYYY.MM.DD}` | 报告日期 |
| `{平台名}` | PC / android / ios |
| `{N}` | 该平台有执行记录的设备总数 |
| `{M}` | 执行时长 ≥ 4小时的设备数 |
| `{版本号}` | packageVersion |
| `{任务URL}` | `https://uauto2.testplus.cn/project/jxsj4/taskDetail?taskId={buildId}` |
| `{异常判断}` | `未发现异常` 或 `发现X台异常` |
| `{设备名}` | 设备名称（如：华为P40Pro(TDR)(JXSJ4)(2)(省电)） |
| `{IP}` | 设备IP地址 |
| `{时长}` | 执行时长（如：0小时19分46秒） |
| `{卡死原因}` | 卡死原因描述（如：卡死、资源before_run执行出现卡死、主线AutoRun执行出现卡死） |
| `{系统日志链接}` | 系统日志链接 |
| `{游戏日志链接}` | 游戏日志链接 |
| `{Perfeye链接}` | Perfeye性能报告链接 |
| `{Crasheye链接}` | Crasheye崩溃报告链接 |

## 异常设备链接

- `Crasheye` → 崩溃报告（宕机/闪退）
- `系统日志` → system-log
- `游戏日志` → Player.log / game-log
- `Perfeye` → 性能报告

## 平台排序

PC → android → ios

# 《剑侠世界4》稳定性日报模板

## 输出格式（纯文本格式，无加粗）

```
{YYYY.MM.DD}《剑侠世界4》稳定性汇总

一、{平台名}（共{N}台设备，其中{M}台执行超过4小时）
- （版本：v{版本号}）
  - {序号}.[{任务名称}]({任务URL}) 任务执行汇总：共{N}台设备，其中{M}台设备执行超过4小时，{异常判断}
- （版本：v{版本号}）
  - {序号}.[{任务名称}]({任务URL}) 任务执行汇总：共{N}台设备，其中{M}台设备执行超过4小时，以下{X}台出现卡死和宕机
    - {设备名}（{IP}）执行了{时长}，在{checkpoint_error} [Crasheye]({链接}) | [系统日志]({链接}) | [游戏日志]({链接})

二、{平台名}（共{N}台设备，其中{M}台执行超过4小时）
- （版本：v{版本号}）
  - {序号}.[{任务名称}]({任务URL}) 任务执行汇总：共{N}台设备，其中{M}台设备执行超过4小时，{异常判断}

三、{平台名}（共{N}台设备，其中{M}台执行超过4小时）
- （版本：v{版本号}）
  - {序号}.[{任务名称}]({任务URL}) 任务执行汇总：共{N}台设备，其中{M}台设备执行超过4小时，以下{X}台出现卡死和宕机
    - {设备名}（{IP}）执行了{时长}，在{checkpoint_error} [系统日志]({链接}) | [游戏日志]({链接}) | [Perfeye]({链接})
```

## 格式要点

- 标题：纯文本 `YYYY.MM.DD《剑侠世界4》稳定性汇总`（无加粗、无 ####）
- 平台名：`一、PC`（无加粗）
- 设备数：`共N台设备`（无加粗）
- 版本行：`- （版本：v...）`
- 任务行：`  - 1.[任务名](URL) 任务执行汇总：...`
- 异常设备缩进：`    - 设备名（IP）执行了时长...`
- 无异常：`未发现卡死和宕机`
- 有异常：`以下X台出现卡死和宕机`
- 链接分隔符：` | `（竖线两侧有空格）

## 字段说明

| 字段 | 说明 |
|------|------|
| `{YYYY.MM.DD}` | 报告日期 |
| `{平台名}` | PC / android / ios / Xbox / PS5 |
| `{N}` | 该平台有执行记录的设备总数 |
| `{M}` | 执行时长 ≥ 4小时的设备数 |
| `{版本号}` | packageVersion |
| `{任务URL}` | `https://uauto2.testplus.cn/project/jxsj4/taskDetail?taskId={buildId}` |

## 异常设备链接

- `Crasheye` → 崩溃报告（宕机/闪退）
- `系统日志` → system-log
- `游戏日志` → Player.log / game-log
- `Perfeye` → 性能报告

## 平台排序

PC → android → ios → Xbox → PS5
